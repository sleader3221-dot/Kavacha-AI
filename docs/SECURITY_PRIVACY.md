# Security And Privacy

- No real personal data in the prototype
- FIR numbers are hashed
- Person, phone, bank, UPI, SIM, vehicle, and artifact identifiers are masked
- POCSO and women/children safety categories use aggregate-only display
- Generated ZCQL is read-only validated
- All copilot actions create audit hashes
- No individual criminality prediction
- Human approval required before patrol action
